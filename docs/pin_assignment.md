# ピン配置

## Arduino UNO

| ピン | 接続先 | 用途 |
|---|---|---|
| A0 | MUX SIG | カラーセンサー値入力 |
| A1 | MUX S2 | チャンネル選択 |
| A4/SDA | MPU-6050 SDA | I2C |
| A5/SCL | MPU-6050 SCL | I2C |
| D2 | MUX S0 | チャンネル選択 |
| D4 | MUX S1 | チャンネル選択 |
| D5 | 後輪DCモーター PWM | 速度 |
| D7 | 後輪DCモーター DIR | 方向 |
| D8 | 超音波 TRIG | 出力 |
| D9 | 超音波 ECHO | 入力 |
| D10 | サーボ信号 | 操舵 |

`Servo` ライブラリはUNOのTimer1を使うため、D9/D10をモーターPWMには使いません。

## MUXチャンネル

| CH | 接続先 |
|---|---|
| 0 | カラーセンサー S1 |
| 1 | カラーセンサー S2 |
| 2 | カラーセンサー S3 |
| 3 | カラーセンサー S4 |
| 4-7 | 未使用 |

## 注意

センサー値は実機配線と抵抗値で変わります。`docs/tuning_guide.md` の手順で白線/黒床の実測値を取り、`Config.h` の閾値を調整してください。
