# ピン配置

## Arduino UNOの前提

Arduino UNOで使えるアナログ入力はA0-A5の最大6本です。今回の構成では、カラーセンサー4つをA0-A3へ直接接続し、MPU-6050はA4/A5のI2Cで接続します。

カラーセンサーはRGBのうち **G出力のみ** を使い、白線/黒床の明るさ差を判定材料にします。MUXは使いません。

## Arduinoピン

| ピン | 接続先 | 用途 |
|---|---|---|
| A0 | カラーセンサー S1 G出力 | 前方頂点の白線/黒床判定 |
| A1 | カラーセンサー S2 G出力 | 左側の白線/黒床判定 |
| A2 | カラーセンサー S3 G出力 | 右側の白線/黒床判定 |
| A3 | カラーセンサー S4 G出力 | 後方頂点の白線/黒床判定 |
| A4/SDA | MPU-6050 SDA | I2C |
| A5/SCL | MPU-6050 SCL | I2C |
| D5 | 後輪DCモーター PWM | 速度 |
| D7 | 後輪DCモーター DIR | 方向 |
| D8 | 超音波 TRIG | 出力 |
| D9 | 超音波 ECHO | 入力 |
| D10 | サーボ信号 | 操舵 |

`Servo` ライブラリはUNOのTimer1を使うため、D9/D10をモーターPWMには使いません。

## カラーセンサー配置

センサーは一直線ではなく、進行方向に対してひし形に配置します。

```text
        進行方向
           ↑
           S1
        S2    S3
           S4
```

- S1: 進行方向に対して一番上の頂点。
- S2: 左側。
- S3: 右側。
- S4: 進行方向に対して一番後ろの頂点。

## 注意

センサー値は実機配線、抵抗値、床材、照明で変わります。`docs/tuning_guide.md` の手順で白線/黒床の実測値を取り、`Config.h` の閾値を調整してください。
