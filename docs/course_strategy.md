# コース走行戦略

## コース条件

`mechatro1_week4_mon (1).pdf`の5ページ目を確認した。コースは3600mm x 1800mm程度、ライン幅は20mm。スタート/ゴールは下中央付近にあり、青矢印の走行方向に沿って進む。

主な特徴:

- 20mm幅の黒線。
- スタート直後に下側の直線。
- 右側に大きい連続カーブ。
- 上側に30度程度の折れ線、交差線、分岐のような横線。
- 上中央にX字の交差。
- 左側にS字に近い連続カーブ。
- 複数の障害物がコース上またはライン近くに置かれる。
- ゴールはスタート付近の横棒を越えた場所で停止する想定。

## センサー配置の仮定

S1, S2, S3, S4を左から右に横一列で配置する。

```text
進行方向
   ↑
 S1  S2  S3  S4
 左      中      右
```

- S2/S3を中心追従に使う。
- S1/S4は曲線、交差線、ライン外れ、側線検知に使う。
- 20mmラインに対してセンサー間隔が広すぎると同時検知が難しいため、S2-S3間隔はライン幅より少し広い程度から調整する。

## 走行方針

1. スタート直後は低速でライン追従する。
2. S2/S3のどちらが黒かで微補正する。
3. S1/S4が反応したらカーブまたは横線とみなし、速度を落として補正を強める。
4. 全白になったら急加速せず、直前の補正方向へ低速で旋回探索する。
5. 障害物が近い場合は一度停止し、右横移動を基本に回避する。
6. MPUのroll角が大きい場合は減速補正し、さらに大きい場合は緊急停止する。
7. 4センサー同時黒が一定時間続いた場合だけゴールと判定する。

## コース上の難所への対応

| 難所 | 対応 |
|---|---|
| 右側の大きい連続カーブ | カーブ状態では速度を`CURVE_SPEED`へ落とし、補正ゲインを上げる |
| 上側の横線/交差 | 4センサー同時黒だけでは即停止しない。一定時間継続でゴール判定 |
| X字交差 | S1/S4の反応を側線として無視し、S2/S3中心の進行を優先 |
| 障害物 | 距離が閾値以下なら停止、横移動、ライン復帰 |
| ライン外れ | 全白で探索状態へ入り、直前の誤差方向へ低速旋回 |

## 実機で調整する値

- `BLACK_THRESHOLD`
- `WHITE_THRESHOLD`
- `BASE_SPEED`
- `CURVE_SPEED`
- `RECOVERY_SPEED`
- `OBSTACLE_DISTANCE_CM`
- `TILT_RECOVER_DEG`
- `TILT_STOP_DEG`
- `GOAL_CONFIRM_MS`
- `OBSTACLE_AVOID_MS`
- 4輪の`*_DIRECTION_SIGN`
