# コース走行戦略

## 前提

- コースは白線/黒床。
- S1, S2, S3, S4を左から右に並べる。
- 中央センサーはS2/S3として扱う。
- 前輪サーボで向きを変え、後輪DCモーターで前後に動く。

```text
進行方向
   ↑
 S1  S2  S3  S4
 左      中      右
```

## 直線

S2/S3のどちらかが白線を一定時間検出し続けたら直線と判断し、`STRAIGHT_SPEED` へ滑らかに上げます。急加速は `MAX_SPEED_STEP` で抑えます。

## カーブ

左側S1/S2または右側S3/S4が白線を一定時間検出したらカーブと判断します。`CURVE_SPEED` に落とし、サーボ角を大きめにしますが、`MAX_SERVO_STEP` で急角度変化を避けます。

## ラインロスト

全センサーが黒床/0相当なら、まず低速で直進します。白線を再検出できなければ短時間バックし、直前の誤差方向へ探索します。

## 障害物

超音波センサーが `OBSTACLE_DISTANCE_CM` 以下を検出したら、まず停止します。その後、サーボを切って低速回避します。実機で不安定な場合は停止維持に寄せてください。
