# コース走行戦略

## 前提

- コースは白線/黒床。
- カラーセンサーはRGBのGのみを使い、A0-A3へ直接接続する。
- センサーは一直線ではなく、進行方向に対してひし形に配置する。
- 前輪サーボで向きを変え、後輪DCモーターで前後に動く。

```text
        進行方向
           ↑
           S1
        S2    S3
           S4
```

- S1: 前方頂点。これから踏む白線を早めに見る。
- S2: 左側。左方向へのずれや左カーブの材料にする。
- S3: 右側。右方向へのずれや右カーブの材料にする。
- S4: 後方頂点。車体中心軸が白線上に残っているかを見る。

## 直線

S1またはS4が白線を一定時間検出し続けたら、車体の中心軸が白線上にいるとみなし、直線候補として扱います。`STRAIGHT_CONFIRM_MS` を超えたら `STATE_STRAIGHT_TRACE` に入り、`STRAIGHT_SPEED` へ滑らかに上げます。急加速は `MAX_SPEED_STEP` で抑えます。

## カーブ

S2が白線を一定時間検出したら左側へラインが寄っている、S3が白線を一定時間検出したら右側へラインが寄っていると判断します。S2/S3の滞在時間が `CURVE_CONFIRM_MS` を超えたらカーブ扱いにし、`CURVE_SPEED` に落とします。サーボ角は `MAX_SERVO_STEP` で急変化を避けます。

## ラインロスト

全センサーが黒床/0相当なら、まず低速で直進します。白線を再検出できなければ短時間バックし、直前の誤差方向へ探索します。ひし形配置ではS1とS4が中心軸の確認に効くため、S1/S4のどちらかが復帰したら通常追従へ戻る候補になります。

## 障害物

超音波センサーが `OBSTACLE_DISTANCE_CM` 以下を検出したら、まず停止します。その後、サーボを切って低速回避します。実機で不安定な場合は停止維持に寄せてください。
